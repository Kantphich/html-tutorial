## 1. HTML คืออะไร?
HTML ย่อมาจาก **HyperText Markup Language** ไม่ใช่ภาษาโปรแกรมมิ่ง (Programming Language) แต่เป็น "ภาษามาร์กอัป" (Markup Language) ที่ใช้สำหรับบอกเว็บบราวเซอร์ (เช่น Chrome, Firefox) ว่าโครงสร้างของหน้าเว็บเพจเราประกอบไปด้วยอะไรบ้าง เช่น ตรงไหนคือหัวข้อ ตรงไหนคือย่อหน้า ตรงไหนคือรูปภาพ

## 2. การเตรียมเครื่องมือ
เพื่อให้เราเขียนโค้ดได้ง่ายและมีสีสันสวยงาม เราจะใช้โปรแกรมประเภท Text Editor ที่ชื่อว่า **Visual Studio Code (VS Code)**
1. ไปที่เว็บไซต์ [code.visualstudio.com](https://code.visualstudio.com/)
2. ดาวน์โหลดและติดตั้งโปรแกรมตามระบบปฏิบัติการของคุณ
3. (แนะนำ) เมื่อติดตั้งเสร็จ ให้เปิดโปรแกรม VS Code ขึ้นมา ไปที่เมนู Extensions (รูปสี่เหลี่ยมด้านซ้ายมือ) และค้นหา/ติดตั้ง Extension ชื่อ **Live Server** (ตัวนี้จะช่วยจำลองเซิร์ฟเวอร์ให้เราดูผลลัพธ์หน้าเว็บได้ง่ายขึ้น และรีเฟรชหน้าให้อัตโนมัติเวลาเราเซฟไฟล์)

## 3. Step-by-step: สร้างหน้าเว็บหน้าแรกของคุณ

เอาล่ะ! มาเริ่มลงมือเขียนโค้ดกันเลย ให้ทำตามขั้นตอนต่อไปนี้ทีละสเต็ปนะครับ:

### Step 1: สร้างโฟลเดอร์สำหรับโปรเจกต์
- สร้างโฟลเดอร์ใหม่ในคอมพิวเตอร์ของคุณ ตั้งชื่อว่า `my-first-website`
- เปิดโปรแกรม VS Code ขึ้นมา
- ลากโฟลเดอร์ `my-first-website` ไปปล่อยในหน้าต่างโปรแกรม VS Code (หรือไปที่เมนู `File > Open Folder...` แล้วเลือกโฟลเดอร์นั้น)

### Step 2: สร้างไฟล์ HTML ไฟล์แรก
- ใน VS Code ให้คลิกปุ่ม **New File** (หรือคลิกขวาที่พื้นที่ว่างซ้ายมือ เลือก New File)
- ตั้งชื่อไฟล์ว่า `index.html` (คำว่า index.html เป็นชื่อมาตรฐานที่บราวเซอร์มักจะมองหาเป็นไฟล์แรกเสมอ)

### Step 3: พิมพ์โครงสร้างพื้นฐานของ HTML
ในไฟล์ `index.html` ให้พิมพ์โค้ดด้านล่างนี้ลงไป (พยายามพิมพ์ด้วยตัวเองเพื่อความคุ้นเคยนะครับ):

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>หน้าเว็บของฉัน</title>
</head>
<body>
    
    <!-- เราจะแสดงเนื้อหาหน้าเว็บตรงนี้ -->

</body>
</html>
```

**อธิบายโค้ด:**
- `<!DOCTYPE html>`: เป็นการประกาศให้บราวเซอร์รู้ว่าเรากำลังใช้ HTML5
- `<html>`: เป็นแท็กคลุมเนื้อหาทั้งหมดของหน้าเว็บ
- `<head>`: เป็นส่วนที่เก็บข้อมูลการตั้งค่าต่างๆ ของเว็บ (ผู้ใช้จะไม่เห็นเนื้อหาส่วนนี้บนหน้าจอ) เช่น ชื่อแท็บ (`<title>`)
- `<body>`: **ส่วนสำคัญ!** เนื้อหาทุกอย่างที่เราอยากให้แสดงบนหน้าจอเว็บ (เช่น ข้อความ, รูปภาพ, ปุ่ม) ต้องเขียนไว้ภายในแท็ก `<body>` นี้

### Step 4: เพิ่มข้อความลงไปในหน้าเว็บ
ให้เราเพิ่มข้อความลงไปในระหว่างแท็ก `<body>` กับ `</body>` ดังนี้:

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>หน้าเว็บของฉัน</title>
</head>
<body>
    
    Hello World! ยินดีต้อนรับสู่ HTML
    นี่คือการเริ่มต้นการสร้างหน้าเว็บเพจของฉัน

</body>
</html>
```

### Step 5: บันทึกไฟล์และดูผลลัพธ์!
- กดบันทึกไฟล์ (ไปที่ `File > Save` หรือกดปุ่ม `Ctrl + S` (Windows) / `Cmd + S` (Mac))
- **วิธีดูผลลัพธ์:** 
  - หากคุณติดตั้ง Live Server ไว้ ให้คลิกขวาที่ไฟล์ `index.html` แล้วเลือก **"Open with Live Server"** หน้าเว็บจะเด้งขึ้นมาในบราวเซอร์อัตโนมัติ
  - หรืออีกวิธี คือให้ไปที่โฟลเดอร์ `my-first-website` ดับเบิ้ลคลิกที่ไฟล์ `index.html` เพื่อเปิดผ่านบราวเซอร์ได้เลย

🎉 **ยินดีด้วย!** คุณสร้างหน้าเว็บหน้าแรกสำเร็จแล้ว
