# บทที่ 2: การจัดการเนื้อหาและข้อความพื้นฐาน (Basic Text & Content)

หลังจากที่เราสร้างหน้าเว็บเปล่าๆ กันไปแล้วในบทที่ 1 ตอนนี้เราจะมาเรียนรู้วิธีการใส่เนื้อหาและจัดการข้อความให้มีรูปแบบต่างๆ กันครับ

## แท็ก (Tags) คืออะไร?
ใน HTML เราจะใช้ **แท็ก (Tags)** เป็นตัวบอกว่าเนื้อหานั้นๆ คืออะไร โครงสร้างของแท็กส่วนใหญ่จะมี **แท็กเปิด** และ **แท็กปิด** คลุมเนื้อหาไว้ตรงกลาง ตัวอย่างเช่น:
`<ชื่อแท็ก> เนื้อหาของเรา </ชื่อแท็ก>`

---

## Step-by-step: จัดการเนื้อหาข้อความบนหน้าเว็บ

เปิดไฟล์ `index.html` ของคุณต่อจากบทที่ 1 และแก้ไขโค้ดภายในแท็ก `<body>` ไปพร้อมๆ กันเลยครับ

### Step 1: หัวข้อ (Headings)
HTML มีแท็กสำหรับสร้างหัวข้อ 6 ระดับ ตั้งแต่ `<h1>` (ใหญ่สุด/สำคัญสุด) ไปจนถึง `<h6>` (เล็กสุด) 

ให้แก้ไขโค้ดใน `<body>` เป็นดังนี้:

```html
<body>
    
    <h1>ยินดีต้อนรับสู่เว็บไซต์ของฉัน</h1>
    <h2>นี่คือหัวข้อรองระดับ 2</h2>
    <h3>นี่คือหัวข้อรองระดับ 3</h3>

</body>
```
*ลอง Save แล้วดูผลลัพธ์ในบราวเซอร์ครับ จะเห็นว่าขนาดตัวอักษรแตกต่างกัน*

**เคล็ดลับ:** ใน 1 หน้าเว็บ **ควรมี `<h1>` เพียงอันเดียวเท่านั้น** เพื่อประโยชน์ต่อการทำ SEO (Search Engine Optimization)

### Step 2: ย่อหน้า (Paragraphs)
ถ้าเราต้องการเขียนข้อความบรรยายความยาวหลายๆ บรรทัด เราจะใช้แท็ก `<p>` (มาจากคำว่า Paragraph)

ให้เพิ่มแท็ก `<p>` ลงไปต่อจากหัวข้อ:

```html
<body>
    
    <h1>ยินดีต้อนรับสู่เว็บไซต์ของฉัน</h1>
    
    <p>สวัสดีครับ เว็บไซต์นี้สร้างขึ้นมาเพื่อฝึกฝนการเขียน HTML 
    และเตรียมความพร้อมสำหรับการทำ Automate Test ด้วย Cypress ในอนาคต</p>
    
    <p>การแยกย่อหน้า จะช่วยให้ผู้ใช้งานอ่านข้อความบนเว็บของเราได้ง่ายขึ้น</p>

</body>
```

### Step 3: การขึ้นบรรทัดใหม่ และเส้นคั่น
- หากต้องการขึ้นบรรทัดใหม่ โดยไม่สร้างย่อหน้าใหม่ ให้ใช้แท็ก `<br>` (Break)
- หากต้องการขีดเส้นคั่นแนวนอน ให้ใช้แท็ก `<hr>` (Horizontal Rule)
*(ข้อสังเกต: 2 แท็กนี้ **ไม่มีแท็กปิด** เราสามารถเขียนลอยๆ ได้เลย)*

ลองเพิ่ม `<hr>` และ `<br>` ดูครับ:

```html
<body>
    
    <h1>ยินดีต้อนรับสู่เว็บไซต์ของฉัน</h1>
    <hr> <!-- เส้นคั่นหน้าเว็บ -->
    
    <p>
        ชื่อ-นามสกุล: จอห์น โด <br> <!-- ขึ้นบรรทัดใหม่ -->
        อายุ: 25 ปี <br>
        อาชีพ: นักพัฒนาซอฟต์แวร์
    </p>

</body>
```

### Step 4: การเน้นข้อความ (Text Formatting)
เราสามารถทำตัวหนาและตัวเอียงได้โดยใช้แท็กต่อไปนี้:
- `<strong>` หรือ `<b>` : ทำตัวอักษรให้เป็น **ตัวหนา** (แนะนำให้ใช้ strong เพราะเน้นความสำคัญเชิงความหมาย)
- `<em>` หรือ `<i>` : ทำตัวอักษรให้เป็น *ตัวเอียง* (แนะนำให้ใช้ em)

```html
<body>
    <!-- ตัดโค้ดเก่าออกก่อน เพื่อความสะอาดของไฟล์ -->
    <h1>ประวัติส่วนตัวของฉัน</h1>
    <p>ฉันชื่อ <strong>จอห์น โด</strong> ปัจจุบันทำงานเป็น <em>นักพัฒนาซอฟต์แวร์</em></p>
</body>
```

### Step 5: การสร้างรายการ (Lists)
การทำเป็นข้อๆ หรือ List จะมี 2 แบบหลักๆ:
1. **แบบไม่มีลำดับ (Unordered List):** ใช้แท็ก `<ul>` คลุม และใช้ `<li>` สำหรับแต่ละข้อ (จะได้เป็นจุดไข่ปลา)
2. **แบบมีลำดับ (Ordered List):** ใช้แท็ก `<ol>` คลุม และใช้ `<li>` สำหรับแต่ละข้อ (จะได้เป็นตัวเลข 1, 2, 3...)

ลองเพิ่มรายการสิ่งที่ชอบทำลงไป:

```html
<body>
    <h1>งานอดิเรกของฉัน</h1>
    
    <!-- รายการแบบไม่มีลำดับ (จุดดำ) -->
    <ul>
        <li>อ่านหนังสือ</li>
        <li>เขียนโปรแกรม</li>
        <li>เล่นเกม</li>
    </ul>

    <h2>เป้าหมายในปีนี้</h2>
    
    <!-- รายการแบบมีลำดับ (ตัวเลข) -->
    <ol>
        <li>เรียนรู้ HTML ให้จบ</li>
        <li>เริ่มศึกษา Cypress Testing</li>
        <li>เขียน Automated Test ให้ได้</li>
    </ol>
</body>
```

### Step 6: Cypress Focus - การกำหนดคลาสและไอดีพื้นฐาน (Class & ID)
สำหรับคนที่ปูพื้นฐานไปเขียนเทส การอ้างอิง Element เป็นสิ่งสำคัญมาก เราสามารถเพิ่ม Attribute `id` (ห้ามซ้ำ) และ `class` (ซ้ำได้) เข้าไปในแท็กต่างๆ ได้

ลองเพิ่ม id และ class ในโค้ด:

```html
<body>
    <h1 id="page-title">รายการช้อปปิ้ง</h1>
    
    <ul>
        <li class="shopping-item">นมสด</li>
        <li class="shopping-item">ขนมปัง</li>
        <li class="shopping-item">ไข่ไก่</li>
    </ul>
</body>
```
*ใน Cypress เราอาจจะเขียนเทสว่าให้ไปค้นหา Element ที่มี `id="page-title"` ว่าแสดงผลถูกต้องหรือไม่*

## สรุปบทที่ 2
ตอนนี้คุณสามารถจัดการเนื้อหาบนหน้าเว็บได้แล้ว!
- `<h1>` ถึง `<h6>` สำหรับสร้างหัวข้อ
- `<p>` สำหรับเขียนย่อหน้าข้อความ
- `<br>` และ `<hr>` สำหรับจัดบรรทัดและขีดเส้นแบ่ง
- `<strong>` และ `<em>` สำหรับเน้นข้อความ
- `<ul>`, `<ol>`, `<li>` สำหรับการสร้างรายการ

> ในบทต่อไป เราจะไปดูการเพิ่ม **"รูปภาพ"** และสร้าง **"ลิงก์"** เชื่อมโยงไปมากันครับ!
