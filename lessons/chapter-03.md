# บทที่ 3: การเชื่อมโยงลิงก์และการใส่รูปภาพ (Links & Images)

เว็บไซต์ที่ดีจะขาดการเชื่อมโยงข้อมูลไปยังหน้าต่างๆ และการใส่รูปภาพเพื่อความสวยงามไปไม่ได้ ในบทนี้เราจะมาเรียนรู้การสร้าง Link และการแสดงผลรูปภาพกันครับ

---

## Step-by-step: การเพิ่มลิงก์และรูปภาพ

ยังคงเปิดไฟล์ `index.html` ไว้เหมือนเดิม แล้วทำตามทีละขั้นตอนกันเลย

### Step 1: การสร้างลิงก์ (Links)
เราใช้แท็ก `<a>` (Anchor tag) สำหรับสร้างการเชื่อมโยง โดยเราต้องบอกเบราว์เซอร์ด้วยว่าเมื่อคลิกแล้วจะให้พาไปที่ไหน ผ่าน Attribute ที่ชื่อว่า `href`

รูปแบบ: `<a href="URL_ปลายทาง">ข้อความที่จะให้ผู้ใช้คลิก</a>`

ให้ลองพิมพ์โค้ดด้านล่างลงไปใน `<body>`:

```html
<body>
    <h1>เว็บไซต์เพื่อนบ้านของฉัน</h1>
    
    <p>
        หากต้องการค้นหาข้อมูล ลองไปที่เว็บ 
        <a href="https://www.google.com">Google</a> ดูสิ!
    </p>
</body>
```
*ลองคลิกคำว่า Google ในหน้าเว็บของคุณดู มันจะเปลี่ยนหน้าไปยังเว็บ Google ทันที*

### Step 2: การเปิดลิงก์ในแท็บใหม่ (Open in New Tab)
ถ้าสังเกตจาก Step 1 เมื่อเราคลิกลิงก์ หน้าเว็บของเราจะหายไปแล้วแทนที่ด้วย Google หากเราต้องการให้เปิดใน "แท็บใหม่" (New Tab) เราจะใส่ Attribute ที่ชื่อว่า `target="_blank"` เพิ่มเข้าไป

```html
<body>
    <p>
        คุณสามารถเรียนรู้เรื่อง Cypress เพิ่มเติมได้ที่
        <a href="https://www.cypress.io/" target="_blank">เว็บไซต์ของ Cypress</a>
    </p>
</body>
```

### Step 3: การใส่รูปภาพ (Images)
เราใช้แท็ก `<img>` ในการแสดงรูปภาพ ซึ่งแท็กนี้เป็นแท็กที่ **ไม่มีแท็กปิด**
เราจะต้องระบุที่อยู่ของรูปภาพผ่าน Attribute `src` (Source) และคำอธิบายรูปภาพผ่าน `alt` (Alternative Text)

รูปแบบ: `<img src="ที่อยู่รูปภาพ" alt="คำอธิบายรูปภาพ">`

**การใส่รูปภาพจาก Internet (URL):**
```html
<body>
    <h2>รูปภาพจากอินเทอร์เน็ต</h2>
    
    <img src="https://images.unsplash.com/photo-1542831371-29b0f74f9713?w=400" alt="หน้าจอคอมพิวเตอร์เขียนโค้ด">
</body>
```

### Step 4: การปรับขนาดรูปภาพ
หากรูปภาพมีขนาดใหญ่เกินไป เราสามารถใช้ Attribute `width` (ความกว้าง) และ `height` (ความสูง) มาช่วยกำหนดขนาดได้ (ส่วนใหญ่กำหนดแค่ `width` อย่างเดียว รูปจะปรับสัดส่วนอัตโนมัติ)

```html
<body>
    <h2>การปรับขนาดรูปภาพ</h2>
    
    <img src="https://images.unsplash.com/photo-1542831371-29b0f74f9713" alt="หน้าจอคอมพิวเตอร์เขียนโค้ด" width="300">
</body>
```

### Step 5: Cypress Focus - ทำไม `alt` ถึงสำคัญ?
`alt` (Alternative Text) มีไว้สำหรับอธิบายรูปภาพให้ผู้พิการทางสายตา (Screen Readers) ได้รับรู้ และจะแสดงขึ้นมาเมื่อโหลดรูปไม่สำเร็จ 

นอกจากนี้ **ในการเขียน Test ด้วย Cypress** เรามักจะใช้ `alt` เพื่อเช็คว่ารูปภาพที่เราต้องการถูกแสดงขึ้นมาถูกต้องหรือไม่

```html
<body>
    <!-- โค้ดตัวอย่างเพื่อให้ Cypress เช็ค -->
    <img src="logo.png" alt="Company Logo" id="main-logo">
</body>
```
*ในเคสเทส เราอาจจะเขียนสั่งให้ Cypress เช็คว่ารูปภาพที่มี `alt="Company Logo"` ต้องปรากฏอยู่บนหน้าเว็บ เป็นต้น*

### Step 6: รูปภาพที่เป็นลิงก์
เราสามารถนำแท็ก `<img>` ไปไว้ข้างในแท็ก `<a>` เพื่อสร้าง "รูปภาพที่คลิกได้"

```html
<body>
    <h2>คลิกที่รูปภาพเพื่อไปยังเว็บไซต์</h2>
    
    <a href="https://www.google.com" target="_blank">
        <img src="https://www.google.com/images/branding/googlelogo/1x/googlelogo_color_272x92dp.png" alt="โลโก้ Google" width="200">
    </a>
</body>
```

---

## สรุปบทที่ 3
- `<a>` ร่วมกับ `href` เพื่อสร้างลิงก์ และใช้ `target="_blank"` เพื่อเปิดแท็บใหม่
- `<img>` ร่วมกับ `src` เพื่อนำรูปภาพมาแสดง
- อย่าลืมใส่ `alt` แอตทริบิวต์เสมอ เพื่อประโยชน์ต่อ Accessibility และการเขียน Automated Test
