# บทที่ 7: ตกแต่งหน้าเว็บด้วย CSS เบื้องต้น (Basic CSS Styling)

HTML คือ "โครงสร้าง" ส่วน CSS (Cascading Style Sheets) คือ "ความสวยงาม" (สี, ขนาด, ตำแหน่ง)
ในบทนี้เราจะมาทำให้เว็บของเราดูน่าสนใจมากขึ้นกันครับ

---

## 1. วิธีการเรียกใช้ CSS
การเขียน CSS ลงในหน้าเว็บมี 3 วิธีหลักๆ
1. **Inline CSS:** เขียนแทรกเข้าไปในแท็ก HTML เลย (เหมาะกับการแก้จุดเล็กๆ)
2. **Internal CSS:** เขียนไว้ในแท็ก `<style>` ที่อยู่ใน `<head>` ของไฟล์นั้น (เหมาะกับเว็บหน้าเดียว)
3. **External CSS:** เขียนแยกเป็นอีกไฟล์หนึ่ง นามสกุล `.css` แล้วดึงเข้ามาใช้ (ดีที่สุดและนิยมที่สุด!)

ในบทนี้เพื่อความรวดเร็ว เราจะใช้แบบที่ 2 (Internal CSS) นะครับ

---

## Step-by-step: แต่งหน้าเว็บเบื้องต้น

เปิดไฟล์ `index.html` ของคุณ แล้วล้างโค้ดใน `<head>` และ `<body>` ให้เป็นแบบฟอร์มด้านล่างนี้ก่อนครับ

### Step 1: เตรียมโครงสร้าง HTML
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>หน้าเว็บตกแต่งด้วย CSS</title>
    
    <!-- เราจะเขียน CSS ในแท็ก style นี้ -->
    <style>
        
    </style>
</head>
<body>
    
    <header>
        <h1>ยินดีต้อนรับสู่เว็บที่มีสีสัน!</h1>
    </header>

    <main>
        <p class="highlight-text">เนื้อหาส่วนนี้มีความสำคัญมาก</p>
        <p>ส่วนนี่คือย่อหน้าปกติธรรมดา</p>
        
        <button id="primary-btn">คลิกฉันสิ</button>
    </main>

</body>
</html>
```

### Step 2: การอ้างอิง (Selectors)
การเขียน CSS เราต้อง "อ้างอิง" (Select) ว่าเราต้องการเปลี่ยนหน้าตาของชิ้นส่วนไหนใน HTML
- **อ้างอิงด้วยชื่อแท็ก (Tag):** เปลี่ยนทุกอันที่เป็นแท็กนั้น
- **อ้างอิงด้วยคลาส (Class):** ใช้จุด `.` นำหน้า
- **อ้างอิงด้วยไอดี (ID):** ใช้เครื่องหมาย `#` นำหน้า (มีได้แค่อันเดียวใน 1 หน้า)

ให้พิมพ์ CSS เหล่านี้ลงไปในระหว่างแท็ก `<style>` กับ `</style>`

```css
        /* 1. อ้างอิงแท็ก: เปลี่ยนพื้นหลังและสีตัวอักษรของทั้งหน้า */
        body {
            background-color: #f4f4f9; /* สีเทาอ่อน */
            font-family: Arial, sans-serif; /* เปลี่ยนรูปแบบฟอนต์ */
            color: #333333;
        }

        /* 2. อ้างอิงแท็ก: เปลี่ยนสีและจัดกึ่งกลางให้ h1 */
        h1 {
            color: #0056b3; /* สีน้ำเงิน */
            text-align: center;
        }

        /* 3. อ้างอิงคลาส: เปลี่ยนเฉพาะ <p> ที่มี class="highlight-text" */
        .highlight-text {
            background-color: yellow;
            font-weight: bold;
        }

        /* 4. อ้างอิงไอดี: เปลี่ยนเฉพาะปุ่มที่มี id="primary-btn" */
        #primary-btn {
            background-color: #28a745; /* สีเขียว */
            color: white;
            border: none;
            padding: 10px 20px; /* ระยะห่างในปุ่ม (แนวตั้ง แนวนอน) */
            cursor: pointer; /* เปลี่ยนเมาส์เป็นรูปมือเมื่อชี้ */
        }
```
*ลองเซฟไฟล์แล้วไปดูที่เบราว์เซอร์ครับ! หน้าเว็บจะดูสวยงามและเป็นระเบียบขึ้นทันตาเห็น*

### Step 3: ทำความรู้จัก Box Model (ระยะห่าง)
สิ่งสำคัญที่สุดในการจัด Layout คือ Box Model ซึ่งประกอบด้วย 3 ส่วนหลัก:
- **Padding:** ระยะห่างภายใน (ดันเนื้อหาออกจากขอบ)
- **Border:** เส้นขอบ
- **Margin:** ระยะห่างภายนอก (ดันกล่องเราให้ห่างจากกล่องคนอื่น)

ลองเพิ่ม Margin เข้าไปที่ปุ่มในไฟล์ CSS เพื่อดันปุ่มให้ห่างจากข้อความด้านบนครับ:
```css
        #primary-btn {
            background-color: #28a745;
            color: white;
            border: none;
            padding: 10px 20px;
            cursor: pointer;
            margin-top: 20px; /* เพิ่มบรรทัดนี้: ดันปุ่มให้ห่างจากด้านบน 20px */
            border-radius: 5px; /* เพิ่มบรรทัดนี้: ทำให้ขอบปุ่มโค้งมนมน */
        }
```

## สรุปบทที่ 7
- CSS ช่วยตกแต่งหน้าเว็บ
- `Tag Selector` ใช้จัดรูปแบบภาพรวม (เช่น `body {}`)
- `Class Selector` ขึ้นต้นด้วย `.` ใช้จัดรูปแบบสิ่งที่ใช้ซ้ำๆ (เช่น `.highlight {}`)
- `ID Selector` ขึ้นต้นด้วย `#` ใช้จัดรูปแบบของที่มีชิ้นเดียวเฉพาะเจาะจง (เช่น `#main-btn {}`)
- `padding` คือระยะห่างในกรอบ ส่วน `margin` คือระยะห่างนอกกรอบ
